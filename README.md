# ✈️ Airline Management System

A modern desktop-based Airline Management System developed using **C++/CLI**, **Windows Forms**, and **.NET Framework**. The application provides a centralized platform for managing airline operations, including passenger records, flight scheduling, reservations, and user authentication.

---

## 📌 Overview

The Airline Management System is designed to streamline airline administration through an intuitive desktop interface. It enables efficient management of flights, passengers, bookings, and user accounts while demonstrating object-oriented programming concepts and desktop application development practices.

This project was developed to showcase desktop application development using C++/CLI and Windows Forms within the Microsoft .NET ecosystem.

## 🚀 Getting Started

### Prerequisites

* Windows Operating System
* Visual Studio 2019 or later
* .NET Framework 4.7.2 or higher

### Installation

Clone the repository:

```bash
git clone https://github.com/Nain-007sh/airline-management-system-cpp-cli.git
```

Open the solution in Visual Studio and build the project:

```bash
Build → Build Solution
```

Run the application:

```bash
Debug → Start Without Debugging
```
